# 3D Portfolio - Ganesh KS

A beautiful 3D portfolio website featuring an isometric room with interactive project displays, built with Three.js.

**Inspired by:** [Rachel Wei's Portfolio](https://rachelqrwei.ca/)

## Features âœ¨

- **Isometric 3D Room** - Cozy bedroom/workspace environment with floating platform
- **Interactive Menu** - 3D text menu with About Me, Projects, Work Experience, Education, and Contact sections
- **Scrapbook About Me** - Polaroid-style photo gallery with personal information
- **Desktop Projects** - Folder-style project showcase with mailbox opening animation
- **Timeline Work Experience** - Professional timeline with achievements and skills
- **ID Card Education** - Student ID card style education display with clickable certificates
- **Business Card Contact** - Professional contact card with social links
- **Music Player** - Background music with play/pause and volume controls (2 loops only)
- **Reflective Floor** - Mirror-like floor effect for enhanced visuals
- **Warm Lighting** - Cozy atmosphere with multiple light sources
